# SPAdes

SPAdes (St. Petersburg genome assembler) is a genome assembly toolkit containing
various assembly pipelines. It supports Illumina reads, PacBio CLR and Oxford
Nanopore reads, as well as hybrid assemblies.

**Homepage:** https://github.com/ablab/spades

## Available Versions

- **Version 3.15.5**

## Binaries

```spades.py metaspades.py plasmidspades.py rnaspades.py truspades.py```
