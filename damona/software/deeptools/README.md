# deeptools

deeptools is a suite of Python tools particularly developed for the efficient
analysis of high-throughput sequencing data, such as ChIP-seq, ATAC-seq and
RNA-seq.

**Homepage:** https://deeptools.readthedocs.io/

## Available Versions

- **Version 3.5.4**
